## 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Project Overview
KeenKeeper is a modern web application designed to help users maintain meaningful relationships with their friends. It allows users to track interactions, set communication goals, and stay connected through an intuitive and visually appealing interface.

## 🛠️ Technologies Used
- Next.js (App Router)
- React.js
- Tailwind CSS
- React Icons
- React Toastify
- Recharts (for analytics charts)
- Context API (for global state management)

## 🌟 Key Features
### 1. 📋 Friend Management System
 Users can view detailed friend profiles including status, tags, contact goals, and personal information

 ### 2. ⚡ Quick Interaction Logging
 Easily log Call, Text, or Video interactions with a single click, automatically updating the Timeline.

 ### 3. 📊 Analytics Dashboard
 Visualize communication patterns using charts to better understand and improve relationships.
